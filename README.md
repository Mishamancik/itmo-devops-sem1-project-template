# Финальный проект 1 семестра

Задание для простого уровня: REST API сервис для загрузки и выгрузки данных о ценах.


## Требования к системе

Для локального запуска приложения вам понадобятся: GitHub, Git, Docker + PostgreSQL-контейнер, Go версии 1.23.3.
    

## Установка и запуск

Чтобы установить и запустить приложение локально:

- Сделайте fork этого репозитория к себе в GitHub
- Сделайте git clone проекта к себе на рабочую машину
- Запустите в docker контейнер postgresql специальной командой (приведена ниже)
- Запустите ./scriptsp/repare.sh , находясь в корне проекта
- Запустите ./scripts/run.sh , находясь в корне проекта
- После сообщения о запуске сервера на порту 8080 можно тестировать приложение
- Для этого сделайте curl-запрос (приведен ниже) или выполните последний шаг 
- Запуститите ./scripts/tests.sh 1 , находясь в корне проекта 

Команда для запуска контейнера postgresql:
docker run --name project-sem-1-db \
  -e POSTGRES_USER=validator \
  -e POSTGRES_PASSWORD=val1dat0r \
  -e POSTGRES_DB=project-sem-1 \
  -p 5432:5432 \
  -d postgres:15

Пример Post-запроса:
  curl -X POST -F "file=@sample_data.zip" http://localhost:8080/api/v0/prices

Пример Get-запроса:
  curl -X GET http://localhost:8080/api/v0/prices -o result.zip
